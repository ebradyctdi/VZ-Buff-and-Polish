Setup Instructions

To set up the Google Sheet backend:

  1. Create a new Google Sheet
  2. Add a tab called "Device Information" with headers: IMEI, OEM, SKU, MODEL, Order Type, DISPO IN
  3. Paste your device data into that tab (IMEI in column A)
  4. Go to Extensions → Apps Script
  5. Paste the contents of google-apps-script.js and save
  6. Click Deploy → New Deployment → Web App
  7. Set "Execute as" to Me, "Who has access" to Anyone
  8. Copy the Web App URL and paste it above
